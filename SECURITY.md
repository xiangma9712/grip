# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within `grip`, please **do not** open a public issue.
Instead, please report it via [GitHub Private Vulnerability Reporting](https://github.com/xiangma9712/grip/security/advisories/new) or directly contact the maintainers.

We will acknowledge your report within 48 hours and provide updates on resolution progress.
